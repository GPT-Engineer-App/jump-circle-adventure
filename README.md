# jump-circle-adventure

A 2d game where you are a circle that needs to jump over spikes to not die and avoid obstacles. The finish line is black and white. Steering possible on a mobile phone.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/jump-circle-adventure.git
cd jump-circle-adventure
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
